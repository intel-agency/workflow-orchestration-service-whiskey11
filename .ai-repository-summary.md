# Repository Summary: workflow-orchestration-service-whiskey11

## Overview
Standalone client/server orchestration service that migrates orchestration from GitHub Actions to a self-hosted model.

## Repository
- **Owner**: intel-agency
- **Name**: workflow-orchestration-service-whiskey11
- **Default Branch**: main

## Tech Stack
- Python 3.12+, FastAPI, httpx, Pydantic, uvicorn
- Docker, Docker Compose
- opencode CLI (agent runtime)
- uv (Python package manager)

## Key Directories
| Directory | Purpose |
|-----------|---------|
| `client/src/` | Orchestration client Python modules |
| `client/scripts/` | Client-side shell scripts |
| `plan_docs/` | Project planning documents |
| `scripts/` | Shared scripts (shell bridge, validation) |
| `test/` | Test suite (bash + Pester) |
| `.opencode/` | Agent definitions and commands |
| `docs/` | Documentation |
| `.github/` | GitHub workflows and config |

## Build & Run
- Install: `cd client && uv pip install -e .`
- Validate: `pwsh -NoProfile -File ./scripts/validate.ps1 -All`
- Docker: `docker compose up`
